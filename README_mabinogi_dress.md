# 🧥 마비노기 의장 대시보드

> 마비노기 외형 BM 담당자를 위한 의장 데이터 관리 및 판매 전략 분석 툴

**[→ 라이브 데모 보기](https://jiye0813.github.io/mabinogi-dress/)**

---

## 만든 이유

마비노기 의장은 확률형 구조로 판매되는 외형 BM의 핵심 상품군입니다. 판매 전략을 세울 때는 단순한 아이템 정보뿐 아니라 **어떤 의장이 재판매 시점에 왔는지**, **어떤 유형이 수요가 높은지**, **지금 팔지 않고 있는 의장 중 재출시 가치가 있는 것은 무엇인지**를 한눈에 파악해야 합니다.

이 툴은 해당 판단에 필요한 데이터를 직접 입력하고, 분석적 관점에서 시각화하여 담당자의 의사결정을 지원합니다.

---

## 주요 기능

### 대시보드 — 판매 전략 인사이트

| 뷰 | 설명 |
|-----|------|
| 판매 검토 필요 의장 | 한 번도 팔지 않은 의장(미판매) 또는 마지막 판매 후 180일 초과 의장 목록 |
| 재판매 주기 분석 | 판매 이력 2회 이상 의장의 평균 재판매 주기를 계산해 다음 판매 예상 시점 표시 |
| 재출시 후보 의장 | 판매 이력 있음 + 현재 미판매 중인 의장을 판매 횟수 순으로 정렬 — 재출시 검토 리스트 |
| 수요 분포 | 스타일 / 성별 / 종족별 판매 비중 바 차트 |
| 시즌별 판매 집중도 | 시즌 태그 기준 누적 판매 이력 합산 — 어떤 시즌 컨셉이 반복 판매되는지 파악 |

### 의장 전체보기
- 성별 · 종족 · 종류 · 스타일 · 시즌 · 판매 상태 기준 다중 필터
- 테이블 뷰 / 그리드 뷰 전환
- 클릭 시 상세 모달 (이미지, 스탯, 색상 파트, 인챈트, 경매장 시세, 판매 이력 타임라인)

### 판매 이력 상세 기록
각 판매 회차마다 아래 항목을 입력할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 시작일 / 종료일 | 판매 기간 (자동으로 기간(일) 계산) |
| 판매가(원) | 해당 회차의 실제 판매 가격 |
| 등장확률(%) | 확률형 아이템의 등장 확률 |
| 판매 채널 | 캐시샵 / 이벤트 / 패키지 |
| 비고 | 콜라보, 특이사항 등 자유 메모 |

판매가와 확률을 함께 기록하면 **확률 대비 효율 및 가치 검증**이 가능합니다.

### Admin — 데이터 편집
- 의장 등록 · 수정 · 삭제
- 색상 파트 / 인챈트 / 세트 아이템 정보 관리
- 경매장 시세 (현재 / 최저 / 최고) 기록
- JSON 내보내기 / 가져오기

## 포트폴리오 관점에서

이 툴은 단순한 데이터 관리 도구가 아닙니다.

- **BM 기획 경험 기반 설계** — 마비노기 확률형 의장의 판매 구조를 직접 담당한 경험을 바탕으로, 실무에서 실제로 필요한 판단 기준(재판매 주기, 판매 공백, 재출시 가치)을 툴로 구현했습니다.
- **데이터 → 인사이트 → 액션** 흐름을 한 화면에서 지원합니다.
- [BM 매출 예측 시뮬레이터](https://jiye0813.github.io/bm-simulator/)와 함께, **분석(드레스룸) → 기획(시뮬레이터)** 워크플로를 구성합니다.
